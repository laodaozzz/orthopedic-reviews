## English

A curated collection of interactive HTML-based orthopedic medical literature reviews and educational summaries. This repository contains over 600 richly formatted articles covering a wide range of orthopedic surgery topics including sports medicine, trauma, joint replacement, and spine surgery.

### Contents

- **ACL Reconstruction** — evidence reviews, graft choices, femoral tunnel techniques
- **AO Fracture Management** — implant applications, preoperative planning guidelines
- **Joint-Specific Topics** — AC joint dislocation, hip replacement, knee arthroplasty
- **Biomaterials** — 3D printed biomaterials for bone defects
- **AI in Orthopedics** — research platforms, medical education applications
- **Guidelines & Guidelines** — AAOS, AMC, and other clinical practice summaries
- And hundreds more across the orthopedic literature spectrum

### Format

Each HTML file is a self-contained, beautifully styled educational page with:
- Interactive tables and figures
- Responsive design for desktop and mobile viewing
- Evidence-based summaries with citations
- Clinical pearls and key takeaways

### Usage

Clone the repository and open any `.html` file in your browser:

```bash
git clone https://github.com/laodaozzz/orthopedic-reviews.git
cd orthopedic-reviews
# Open any file in your browser
```

Alternatively, serve locally with any HTTP server:

```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## 中文

精选骨科医学文献综述与交互式教育摘要 HTML 合集。本仓库包含 600 余篇排版精美的文章，覆盖运动医学、创伤骨科、关节置换、脊柱外科等广泛的骨科学术领域。

### 内容分类

- **ACL 重建** — 循证综述、移植物选择、股骨隧道技术
- **AO 骨折管理** — 植入物应用、术前规划指南
- **各关节专题** — 肩锁关节脱位、髋关节置换、膝关节置换
- **生物材料** — 骨缺损 3D 打印生物材料
- **AI 在骨科的应用** — 研究平台、医学教育应用
- **临床指南** — AAOS、AMC 等临床实践总结
- 以及数百篇涵盖骨科文献谱系的其他文章

### 格式说明

每个 HTML 文件都是一个独立的、排版精美的教育页面，包含：
- 交互式表格与图表
- 适配桌面和移动端的响应式设计
- 附带引用的循证摘要
- 临床要点与关键结论

### 使用方式

克隆仓库后在浏览器中打开任意 `.html` 文件：

```bash
git clone https://github.com/laodaozzz/orthopedic-reviews.git
cd orthopedic-reviews
# 在浏览器中打开任意文件
```

也可通过 HTTP 服务器本地查看：

```bash
python -m http.server 8000
# 然后访问 http://localhost:8000
```
